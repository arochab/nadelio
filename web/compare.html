How it works

Nadelio is a 4-step pipeline that turns a single brand name into a dual visibility map, SERP and AI:

1. You type a brand name.
2. Claude Haiku infers your sector, identifies 4-5 direct competitors, and generates buyer-intent search queries.
3. Each query runs against live Google via Bright Data, and Claude checks which brands an AI assistant would recommend for the same queries.
4. Results are rendered as a dual heatmap: green for Google SERP ranks, purple for AI engine mentions, so you can see the gap between traditional and AI-powered search.
